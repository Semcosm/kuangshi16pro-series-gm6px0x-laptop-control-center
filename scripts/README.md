# Script Layout

Repository helper scripts are organized by source or transport:

- `scripts/amw0/`: AMW0 WMI transport, WKBC, SCMD, and EC monitoring helpers
- `scripts/ec/`: direct EC observation helpers
- `scripts/acpi/`: ACPI scan or table-oriented helpers
- `scripts/windows/`: Windows-side capture and triage helpers

These scripts are part of the tracked development workspace. Generated outputs
from running them should stay untracked.
